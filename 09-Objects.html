Objects

Object is a Non-Primitive Data-Type

Objects are further divided into three categories:

  1. Object Literal
  2. Arrays
  3. Functions
  • To compare objects, we compare their memory addresses.
  • Whenever we want to compare objects, we compare their addresses.

const car = {type:"Fiat", model:"500", color:"white"}; 
        const myName = 'Chirag';
        const userName1 = ''; // Both have the same memory address
        const userName2 = '';
        
        const user1 = {}; // Different memory addresses
        const user2 = {};

We can access elements of an object in two ways:

  1. Dot notation
  2. Bracket notation
        const product = {                       
            'product-price': '290',
            product_rating: {
                stars: '3.5/5',
                count: 98
            },
            'ramada-price': '$9.89/kg'  // Using bracket notation
        };

        product.transport = 'By-Road';          // Using dot notation
        product['delivery-time'] = '3 days';

Creating an object within a function:

        const product = {
            company_name: 'Safari',
            product_name: '35L Tracking Bag',
            price: `380.20$`,
            more: {
                another_value: 21,
                second_other_value: 23,
                third_value: 'One piece is Real'
            },

            // Adding a function to the object
            fun: function() {
                console.log('One Piece is Real Coming from a function');
            }
        };

        console.log(product.more.third_value); // Accessing an element within an object
        product.fun();  // Calling a function within an object
  • **Special Characters**: You can access properties with spaces or symbols (e.g., `['delivery-time']`).
  • **Dynamic Access**: Use variables or expressions for property names.
  • **Non-String Keys**: Works for numbers or other data types.
  • **Reserved Keywords**: You can access properties like `class`, `export` without errors.

We could also use object nesting:

        const product = {
            company_name: 'Safari',
            product_name: '35L Tracking Bag',
            price: '380.20$',
            rating: {
                stars: 4.4,
                count: 90
            }
        };

Notes:

  • Bracket notation allows access to properties with special characters or spaces, such as `['delivery-time']`.
  • When using bracket notation, you can use variables or expressions to define property names.
  • Inside an object, you can store any type of value, including other objects, arrays, and functions.
  • Bracket notation is helpful for properties like `first-name`, where dot notation might not work properly.
