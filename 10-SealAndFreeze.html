Seal and Freeze

Object.seal(objectName)

  • Object.seal() prevents adding new properties and deleting existing properties of an object.
  • It does not prevent modifying the values of existing properties, as long as they are writable.
  • It works only with the keys (properties) of the object, not the values directly. The ability to modify a value depends on whether the property is writable.

    • After sealing an object:

  • You can change the values of existing properties (if they are writable).
  • You cannot add new properties or delete existing ones.

Object.freeze()

  • Object.freeze() makes an object **immutable**.
  • It prevents **adding**, **deleting**, and **modifying** both the keys and the values of an object.
  • After freezing an object, the properties become **non-writable**, **non-configurable**, and **non-enumerable**.
  • Note: Object.freeze() is shallow, meaning it only freezes the immediate properties of the object, not the nested ones.

Using the in Keyword with Objects

'key of object' in ObjectName
'firstName' in user

It does not check the value, it just checks whether a key exists in the object.

Code Example:

        const user = {
            firstName: 'Agney',
            lastName: 'Patel',
            pata: {
                city: 'Bangalore',
                pincode: '879888',
                state: 'Karnataka',
                moreDetails: {
                    population: 99999999999,
                    area: '798 sq km',
                },
            },
            age: 15,
            isGraduate: false,
        }

        console.log(user.isGraduate);  // Outputs: false

        // Sealing the object
        Object.seal(user);

        // Attempting to add new properties (will fail)
        user.middleName = 'Jasmin';  // This will not work
        console.log(user.middleName);  // Outputs: undefined

        // Attempting to delete a property (will fail)
        delete user.age;  // This will not work
        console.log(user.age);  // Outputs: 15

        // Modifying existing properties (works if writable)
        user.firstName = 'Anjali';  // This will work
        console.log(user.firstName);  // Outputs: Anjali

        // Freezing the object
        Object.freeze(user);

        // Attempting to modify properties (will fail)
        user.firstName = 'Ravi';  // This will not work
        console.log(user.firstName);  // Outputs: Anjali (remains unchanged)
