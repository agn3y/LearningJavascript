Understanding getAttribute and setAttribute

This is a div element. Its current `class` attribute is "originalClass".

To read an attribute, use: element.getAttribute(attributeName)
Example: document.getElementById("exampleDiv").getAttribute("class")
This will return: "originalClass".

To change or set an attribute, use: element.setAttribute(attributeName, value)
Example: document.getElementById("exampleDiv").setAttribute("class", "newClass")
This will change the `class` attribute value of the `

` to: "newClass".

Example Code


    // Access the element
    let div = document.getElementById("exampleDiv");
      
    // Get the current class value
    let currentClass = div.getAttribute("class");
    console.log("Current class:", currentClass);

    // Set a new class value
    div.setAttribute("class", "newClass");
    console.log("New class:", div.getAttribute("class"));

After using setAttribute, the new `class` of the `

` will be "newClass". You can verify the change by reading the attribute again with getAttribute.
