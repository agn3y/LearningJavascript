Basic Array Methods

  1. pop():

    2. -> Removes the last item of an array.

    fruitCollection.pop();
  2. push():

    3. -> Adds one or more items to the end of an array and returns the new length.

    fruitCollection.push('Pineapple');

    -> You can insert multiple items: fruitCollection.push('fruit1', 'fruit2');

  3. shift():

    4. -> Removes the very first item of an array. It shifts all the remaining elements down, which can affect performance for large arrays.

    originalArray.shift();
  4. unshift():

    5. -> Adds new items to the beginning of an array, shifting the existing elements to the right.

    originalArray.unshift(12);
  5. concat():

    6. -> Concatenates (joins) two or more arrays without changing the original arrays.

    originalArray.concat(fruitCollection);
  6. indexOf():

    7. -> Returns the index (position) of the first occurrence of a specified value. Returns -1 if not found.

    originalArray.indexOf(8);

    Note:

    -> In strings, it returns the index of a particular string. In arrays, it finds the position of a specified element.

  7. includes():

    8. -> Checks if a specified element exists in an array. Returns true or false.

    originalArray.includes(2);
  8. reverse():

    9. -> Reverses the order of elements in an array and overwrites the original array.

    originalArray.reverse();
  9. sort():

    10. -> Sorts the elements in an array in alphabetical or numerical order. This method modifies the original array.

    fruitCollection.sort();

    Note:

    -> It sorts strings alphabetically, but it may not sort numbers as expected. For example, sorting numbers as strings results in this order: {0, 01, 2, 12, 3, 23}.

  10. slice():

    11. -> Returns a shallow copy of a portion of an array, as a new array. The start index is included, but the end index is excluded.

    array.slice(start, end);
  11. splice():

    12. -> Adds and/or removes array elements. This method modifies the original array.

    array.splice(index, count, item1, ..., itemX);
