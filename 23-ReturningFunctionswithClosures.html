Returning Functions with Closures

In JavaScript, functions can return other functions. When a returned function references variables from its parent function, it creates a closure. A closure allows the returned function to "remember" the variables of the parent function even after the parent function has finished executing.

Example


function outer() {
    const a = 4
    function parent() {
        const b = 6
        return function add() {
        console.log(a + b)
        }
    }
    return parent()
}
      
const add1 = outer()
      
console.dir(add1);

Simple Closure Explanation

1. `outer` Function

The outer function creates a variable a = 4 and calls another function called parent.

2. `parent` Function

The parent function creates b = 6 and returns a new function called add.

3. Closure Happens

The add function remembers both a (from outer) and b (from parent) even after the functions are done.

4. What `add` Does

When add1() is called, it adds a and b (4 + 6) and prints 10.
