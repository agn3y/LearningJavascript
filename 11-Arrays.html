Arrays

How to define an Array:

const fruitCollection = ['Apple', 'Mango', 'Dates', 'Grapes', 'ORange'];

Basic Array Methods

  1. pop(): Removes the last element from the array.
    2. fruitCollection.pop();
  2. push(): Adds one or more elements to the end of an array.
    3. fruitCollection.push('Papaya');
  3. shift(): Removes the first element of an array.
    4. fruitCollection.shift();
  4. unshift(): Adds one or more elements to the beginning of an array.
    5. fruitCollection.unshift('Banana');
  5. concat(): Merges two or more arrays.
    6. const newFruitCollection = fruitCollection.concat(['Guava', 'Pineapple']);
  6. indexOf(): Returns the first index of a specified element, or -1 if not found.
    7. fruitCollection.indexOf('Mango');
  7. includes(): Checks if an element exists in the array.
    8. fruitCollection.includes('Mango');
  8. reverse(): Reverses the order of elements in an array.
    9. fruitCollection.reverse();
  9. sort(): Sorts the elements of an array.
    10. fruitCollection.sort();
  10. slice(): Returns a shallow copy of a portion of the array.
    11. const slicedArray = fruitCollection.slice(1, 3);
  11. splice(): Changes the contents of an array by removing or replacing existing elements.
    12. fruitCollection.splice(2, 1, 'Kiwi');

To modify/add an item in Array

  • To add a new property, you can use fruitCollection[5] = 'Papaya';
  • Alternatively, use fruitCollection[fruitCollection.length] = 'new-fruit';
  • If you add 'Sapota(chiku)' at fruitCollection[10], all indices between 5 and 10 will be undefined.

Multidimensional Arrays
