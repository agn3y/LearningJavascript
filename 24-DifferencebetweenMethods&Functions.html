Difference Between Methods and Functions in JavaScript

1. Functions

A function is a block of code that performs a specific task. You can call it to run the task whenever you need.

function sayHello() {
  console.log("Hello!");
}

sayHello(); // Calls the function and prints "Hello!"

2. Methods

A method is just a function that is attached to an object. It belongs to an object and is called using that object.

let person = {
  name: "John",
  sayHello: function() {
    console.log("Hello, " + this.name);
  }
};

person.sayHello(); // Calls the method and prints "Hello, John"

Key Difference

A function is independent and can be called by itself.

A method is a function that belongs to an object and is called using that object.

Example

const methodExplain = {
  name : 'Method Explanation',
  add : function(a,b){
    return a + b;
  },
  sub : function(a,b){
    return a - b;
  },
  mul : function(a,b){
    return a * b;
  },
}
console.dir(methodExplain.add(2,312222));
console.dir(methodExplain.sub(88,3));
console.dir(methodExplain. mul(222,3));
