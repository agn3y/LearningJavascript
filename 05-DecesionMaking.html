Decision Making in JavaScript

If Statements

// Simple if statement
if (condition) {
    statement1;
}

// If-else statement
if (condition) {
    statement1;
} else {
    statement2;
}

// Nested if-else statement
if (condition1) {
    if (condition2) {
        // Both condition1 and condition2 are true
        statement1;
    } else {
        // condition1 is true, condition2 is false
        statement2;
    }
} else {
    // condition1 is false
    statement3;
}

Key Notes

  • The if statement will execute only the next line as a part of its block if braces {} are not used.
  • Always use braces for better readability and to avoid unexpected behavior, especially when dealing with multiple statements.

Code Optimization

Code optimization focuses on enhancing the quality and efficiency of the generated machine code. This includes:

  • Reducing execution time by simplifying logic.
  • Minimizing resource usage such as memory or CPU cycles.
  • Improving overall system performance.

Optimized code not only runs faster but is also easier to maintain and debug.
