Map

Map is a function which returns everything in array format.

The .map() method creates a new array by applying a given function to each element of the original array. It does not modify the original array.

Syntax:

let newArray = array.map(function(currentValue, index, array) {
  // Return the new value
});

Parameters:

  • currentValue: The current element being processed in the array.
  • index (optional): The index of the current element.
  • array (optional): The array that .map() was called on.

Example 1: Doubling Numbers

const numbers = [1, 2, 3, 4];
const doubled = numbers.map(num => num * 2);
console.log(doubled); // [2, 4, 6, 8]

Example 2: Extracting Specific Properties

const users = [
  { name: "Alice", age: 25 },
  { name: "Bob", age: 30 },
  { name: "Charlie", age: 35 }
];
const names = users.map(user => user.name);
console.log(names); // ['Alice', 'Bob', 'Charlie']

Example 3: Using Index

const numbers = [1, 2, 3, 4];
const indexedArray = numbers.map((num, index) => num + index);
console.log(indexedArray); // [1, 3, 5, 7]

Filter

What is filter()?

The filter() method in JavaScript creates a new array with all elements that pass a test implemented by the provided function.

This method does not change the original array; it returns a new array with the elements that match the condition.

Basic Syntax:


array.filter(function(element, index, array) {
  // return true or false
});
  • element: The current element being processed in the array.
  • index: The index of the current element being processed (optional).
  • array: The original array that filter() was called on (optional).

Key Points:

  • The filter() method does not modify the original array.
  • It creates a new array with the elements that pass the test (where the callback function returns true).
  • If no elements pass the test, the new array will be empty.
  • The callback function can accept three arguments: the current element, the index, and the original array.

Example 1: Filter Numbers Greater Than 10


const numbers = [5, 12, 8, 130, 44];
const filteredNumbers = numbers.filter(function(num) {
  return num > 10;  // Only keep numbers greater than 10
});
console.log(filteredNumbers); // Output: [12, 130, 44]

Example 2: Filter Adults (Age >= 18)


const users = [
  { name: "John", age: 28 },
  { name: "Jane", age: 15 },
  { name: "Mike", age: 25 },
  { name: "Mary", age: 30 }
];
const adults = users.filter(function(user) {
  return user.age >= 18;  // Only keep users aged 18 or older
});
console.log(adults); 
// Output: [{ name: "John", age: 28 }, { name: "Mike", age: 25 }, { name: "Mary", age: 30 }]

Example 3: Filter Long Words (More Than 5 Letters)


const words = ["apple", "banana", "cherry", "kiwi", "orange"];
const longWords = words.filter(function(word) {
  return word.length > 5;  // Only keep words with more than 5 letters
});
console.log(longWords); // Output: ["banana", "cherry", "orange"]

Understanding the reduce() Method in JavaScript

The reduce() method is used to process an array and return a single output value. It applies a function (callback) to each element, accumulating a result as it iterates through the array.

Syntax

array.reduce(callback, initialValue);

Parameters:

  • callback: A function executed on each element, with the following arguments:
    • accumulator: The result accumulated from previous iterations.
    • currentValue: The current array element.
    • currentIndex (optional): The index of the current element.
    • array (optional): The original array.
  • initialValue (optional): The initial value for the accumulator. If omitted, the first array element is used, and iteration starts from the second element.

Conclusion

1. map

Purpose: Transform each element in an array.

Example: Convert all strings to uppercase.

2. filter

Purpose: Filter elements based on a condition.

Example: Get all even numbers from an array.

3. forEach

Purpose: Perform an action on each element.

Example: Log each element to the console.
