Spread Operator in JavaScript

The spread operator (...) in JavaScript is a powerful tool used to expand elements of an array or object into individual elements or properties. It is commonly used for tasks like cloning arrays/objects, merging them, or passing multiple arguments to functions.

Syntax

...iterable

The iterable can be an array, string, or object.

Use Cases

1. Copying Arrays

The spread operator makes it easy to copy an array without modifying the original one.


const numbers = [1, 2, 3];
const newNumbers = [...numbers]; // Copy
console.log(newNumbers); // [1, 2, 3]

2. Merging Arrays

You can combine two or more arrays into one.


const array1 = [1, 2, 3];
const array2 = [4, 5, 6];
const mergedArray = [...array1, ...array2];
console.log(mergedArray); // [1, 2, 3, 4, 5, 6]

3. Passing Arguments to Functions

The spread operator can pass an array of arguments to a function.


function add(a, b, c) {
    return a + b + c;
}
const Number = [1, 2, 3];
console.log(add(...Number)); // 6

4. Cloning Objects

You can use the spread operator to copy the properties of one object into another.


const person = { name: "John", age: 30 };
const clonedPerson = { ...person }; // Copy
console.log(clonedPerson); // { name: "John", age: 30 }

5. Merging Objects

You can merge two objects into one.


const obj1 = { a: 1, b: 2 };
const obj2 = { c: 3, d: 4 };
const mergedObj = { ...obj1, ...obj2 };
console.log(mergedObj); // { a: 1, b: 2, c: 3, d: 4 }

6. Destructuring with Rest

The spread operator is the opposite of the rest operator, but they look the same. With destructuring, the rest operator can collect the remaining properties or elements.


const [first, ...rest] = [1, 2, 3, 4];
console.log(first); // 1
console.log(rest); // [2, 3, 4]

Notes

  • Spread works on iterables like arrays and strings.
  • For objects, it is shallow, meaning it only copies one level deep.
