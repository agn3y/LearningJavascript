Arguments Keyword

The arguments object is a special variable that is available within all non-arrow functions. It holds the values of all the arguments passed to the function, even if they aren't explicitly named in the function's parameter list.

Key Points about arguments:

  • Availability: It works in regular functions, but not in arrow functions (since arrow functions donâ€™t have their own arguments object).
  • Array-like: arguments is not an array, but it is array-like. It means you can access arguments using indices like an array (e.g., arguments[0], arguments[1]), but it doesn't have array methods like push() or pop().

Example:

        
function myFunction() {
console.log(arguments[0]); // First argument
console.log(arguments[1]); // Second argument
}

myFunction('apple', 'banana');  // Output: apple, banana

Limitations:

It doesn't work with arrow functions: 

        
const myArrowFunction = () => {
console.log(arguments);  // Error: arguments is not defined
};

In modern JavaScript, you can use the rest parameters (...args) for a cleaner way to handle variable arguments: 

        
function myFunction(...args) {
console.log(args[0]); // First argument
console.log(args[1]); // Second argument
}

myFunction('apple', 'banana');  // Output: apple, banana

Rest parameters are preferred in modern code as they are more flexible and easier to work with than arguments.
