Understanding Prompt Input and Type Conversion in JavaScript

In this example, we explore how JavaScript handles inputs from the prompt() function, especially when dealing with numbers. Inputs from a prompt are treated as strings by default, but JavaScript can perform implicit type conversion in some cases.

For example, multiplying width and height strings such as "8" * "5" results in 40 because JavaScript converts them to numbers automatically. However, during addition, the strings are concatenated, producing results like "8" + "5" = "85".

To ensure correct arithmetic operations, we explicitly convert string inputs to numbers using:

  • +prompt() — a shorthand for converting strings to numbers
  • parseInt() — a more flexible method that allows for specifying base values

Example: Handling Prompt Input

Handling Default Values

To provide default values for prompts, we can use the || operator: 

const name = prompt('Enter your name:') || 'Default Name';
const age = +prompt('Enter your age:') || 18;

document.write(`
Name: ${name}
`);
document.write(`
Age: ${age}
`);

Comparing +prompt() and parseInt()

Using +prompt() converts null or empty values to 0, whereas parseInt() will return NaN in such cases. Choose the method that best fits your requirements.
