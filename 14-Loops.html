While Loop

Three types of loop are there:

  1. WhileLoop
  2. ForLoop
  3. DoWhile

While loop

Syntax:

let num = 1;
while(num<=100){
console.log(num);
num++;
}

In a `while` loop, the condition is placed inside `( )`, and the code to execute as long as the condition is true is written inside `{ }`

for loop

Syntax:

for (let p = 0; p <= 1000; p++) {
if (p % 2 === 0) {
console.log(p);
}
}

Do WhileLoop

Syntax:

let i = 10;
do {
console.log(i);
i++
} while(i < 5);

Note:

Now we have gone through all loops we could see our self using for loop more often. Why?

Because it is simple and easy to use, and we can use scope with it, which other loops don't allow

JavaScript Code Examples


// While Loop
console.log('Program started');
let num = 1;
while (num <= 10) {
    console.log(num);
    num++;
}
console.log('Program Died');

// while loop with array
let sixFriends = ['Ayush', 'Anirudh', 'Harmin', 'Guruvesh', 'Viren', 'Wajid', 'Krunal', 'Tumil', 'Ketan', 'Harshal'];
let i = 0;
while (i < sixFriends.length) {
    console.log(i + 1, sixFriends[i] + '-proCoder');
    i++;
}

// for loop to print even numbers
for (let p = 0; p <= 10; p++) {
    if (p % 2 === 0) {
        console.log(p);
    }
}

// Do while loop
let i22 = 10;
do {
    console.log('first line printed, before checking condition');
    console.log(i22);
    i22++;
} while (i22 < 5);
