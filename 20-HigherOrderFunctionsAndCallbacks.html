Higher Order Functions and Callbacks

Passing Functions as Arguments

In JavaScript, functions can be used like regular values. You can give a function as an input to another function. This is called passing a function as an argument.

A higher-order function is simply a function that takes another function as an argument or returns a function

Explanation:

Imagine you have a box (function), and inside this box, you can put another smaller box (another function). This bigger box (higher-order function) can then either open the smaller box and use what's inside (call the function) or give you a new box (return a function).

// A higher-order function that takes another function as an argument
function greetUser(callback) {
console.log("Hello");
callback();  // Call the callback function after greeting
}
                            
// A simple callback function
function sayGoodbye() {
console.log("Goodbye!");
}
                            
// Using the higher-order function
greetUser(sayGoodbye);

More simple version of this code:

// A higher-order function that calls the callback function directly
function greetUser() {
  console.log("Hello");
  sayGoodbye();  // Directly call the callback function
}

// A simple callback function
function sayGoodbye() {
  console.log("Goodbye!");
}

// Using the higher-order function
greetUser();

In the code, the value "Agney" is passed as the name and the sayGoodbye function is passed as the callback. When the greetUser function is called, it first prints "Hello, Agney" using the value of name. After that, it calls the callback function, which is the sayGoodbye function, and prints "Goodbye!". So, the result is that it prints "Hello, Agney" on one line, and "Goodbye!" on the next line

What is a Higher-Order Function?

A higher-order function is a special type of function in JavaScript that can do one or both of these:

  1. Take another function as input (called a callback function)
  2. Return a function as its output

What is a Callback?

A callback is a function you give to another function. The other function will call (or "use") it later

Examples:

function greetUser(callback) {
console.log("Hello");  // Do something first
callback();            // Then call the function passed as an argument
}
          
function sayGoodbye() {
console.log("Goodbye!");
}
          
// Call the higher-order function
greetUser(sayGoodbye);

How does it work:

  • greetUser is the higher-order function because it takes sayGoodbye as input
  • sayGoodbye is the callback function because it is passed to greetUser and called later

Why Use Higher-Order Functions and Callbacks?

  1. Reusability: You can write one function (like greetUser) and use it with many different callback functions
  2. Do things in order: You can do something first, and then run the callback function

Key Points

  1. Higher-order function: A function that works with other functions (takes them as input or returns them)
  2. Callback: A function you pass to another function to use later
