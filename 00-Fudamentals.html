JavaScript Fundamentals

A quick reference for basic information and important concepts in JavaScript.

Key Concepts

  • Script Loading with defer: 
    <script src="script.js" defer></script>
    Allows the HTML to continue parsing and compiling while the script loads in the background. The script will execute after the HTML parsing is complete.
  • Order of Operations:
    • 1. Parentheses: (...)
    • 2. Multiplication (*) and Division (/)
    • 3. Addition (+) and Subtraction (-)
    • 4. Comparison Operators
    • 5. Logical Operators
  • Data Types:
    • Primitive: number, string, boolean, undefined, null, bigint, symbol
    • Non-Primitive: Object
      • Object Literal
      • Arrays
      • Functions

Working with Strings

  • Strings can be written with single quotes (' '), double quotes (" "), or backticks (` ` for template literals).
  • Converting Strings to Numbers:
    • +'100' → 100
    • parseInt('100agney') → 100
  • Converting Numbers to Strings:
    • 100 + '' → '100'
  • String Methods:
    • length: Returns the length of the string.
    • toUpperCase(): Converts to uppercase.
    • toLowerCase(): Converts to lowercase.
    • trim(): Removes unnecessary spaces.
    • includes(): Checks for a substring.

Variable Declaration

  • let: Block-scoped variable.
  • const: Block-scoped constant.
  • var: Function-scoped variable (not recommended).

Execution Process in JavaScript

JavaScript code executes in two stages:

  1. Memory Creation: Variables are declared and initialized with undefined.
  2. Execution: Values are assigned to the variables.

Infinity and NaN

  • Dividing a positive number by a very small number → Infinity.
  • Dividing a negative number by a very small number → -Infinity.
  • Dividing zero by zero → NaN (Not a Number).
