Understanding the forEach Method in JavaScript

The forEach method is used to execute a function for each element in an array. Itâ€™s commonly used to perform actions like logging or updating data.

Example:

In the following example, we have an array of fruits. The forEach method iterates over each item and logs it to the console:

// JavaScript Code:
const fruits = ['apple', 'banana', 'cherry'];

fruits.forEach((fruit, index) => {
  console.log(`Fruit at index ${index}: ${fruit}`);
});

Output in Console:

Fruit at index 0: apple
Fruit at index 1: banana
Fruit at index 2: cherry

Key Points:

  • The forEach method does not return a new array.
  • It cannot break or stop early (unlike a regular for loop).
  • Useful for performing actions on each element, such as logging or modifying data.
  • we can not assign it a return value by default undefined
