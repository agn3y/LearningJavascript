Difference between innerText and innerContent

1. innerText

Gets the visible text inside an element (what you see on the page).

It ignores hidden text or text inside hidden elements.

Example:

Hello World

innerText of above div: "Hello" (because "World" is hidden)

2. innerContent

innerContent is not a standard property in JavaScript.

It might be confused with innerHTML or textContent.

innerHTML: Gives HTML tags and text inside an element.

textContent: Gives only the text (ignores HTML tags and doesn't care about hidden text).
