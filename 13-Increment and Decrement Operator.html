Combined Assignment Operators

Compound (Combined Assignment) Operators

  • += Addition Assignment Operator

    • let a = 5;
    a += 5;
    console.log(a);

  • -= Subtraction Assignment Operator

    • let b = 5;
    b -= 5;
    console.log(b);

  • %= Remainder Assignment Operator

    • let c = 5;
    c %= 5;
    console.log(c);

  • *= Multiplication Assignment Operator

    • let d = 5;
    d *= 5;
    console.log(d);

  • /= Division Assignment Operator

    • let e = 5;
    e /= 5;
    console.log(e);

  • **= Exponential Assignment Operator

    • let f = 5;
    f **= 5;
    console.log(f);

Increment and Decrement Operators

  1. Increment (++)

    2. Addition by 1

    • Postfix Increment

      • let y = 5;
      let x = y++;

      First, return the variable value, then increment it.

    • Prefix Increment

      • let y = 5;
      let x = ++y;

      Increment the variable first, then return the value.

  2. Decrement (--)

    3. Subtraction by 1

    • Postfix Decrement

      • let y = 5;
      let x = y--;

      First, return the variable value, then decrement it.

    • Prefix Decrement

      • let y = 5;
      let x = --y;

      Decrement the variable first, then return the value.

JavaScript Code Examples


// Addition Assignment Operator
let a = 5;
a += 5; // a is now 10
console.log(a); // Output: 10

// Subtraction Assignment Operator
let b = 30;
b -= 5; // b is now 25
console.log(b); // Output: 25

// Remainder Assignment Operator
let c = 500;
c %= 5; // c is now 0 (remainder of 500 divided by 5)
console.log(c); // Output: 0

// Multiplication Assignment Operator
let d = 1;
d *= 5; // d is now 5
console.log(d); // Output: 5

// Division Assignment Operator
let e = 300;
e /= 5; // e is now 60
console.log(e); // Output: 60

// Exponential Assignment Operator
let f = 2;
f **= 2; // f is now 4 (2 to the power of 2)
f **= 2; // f is now 16 (4 to the power of 2)
console.log(f); // Output: 16

// Increment Operators
let x = 5; // Postfix Increment
let x1 = x++; // x1 is 5, x becomes 6
let y = 5; // Prefix Increment
let y1 = ++y; // y1 is 6, y becomes 6
console.log('Post Increment: ' + x1 + ', ' + 'Pre Increment: ' + y1);

// Decrement Operators
let x11 = 5; // Postfix Decrement
let x111 = x11--; // x111 is 5, x11 becomes 4
let y11 = 5; // Prefix Decrement
let y111 = --y11; // y111 is 4, y11 becomes 4
console.log('Post Decrement: ' + x111 + ', ' + 'Pre Decrement: ' + y111);
