Event Listeners Example

Click:

Triggered when the user clicks on the element. The event logs the type (click).

Double-click:

Triggered when the user double-clicks on the element. It logs the event type (dblclick).

Mouse Enter:

Triggered when the mouse pointer enters the element. It logs the event type (mouseenter).

Mouse Leave:

Triggered when the mouse pointer leaves the element. It logs the event type (mouseleave).

Mouse Move:

Triggered when the mouse pointer moves over the element. It logs the event type (mousemove).

Mouse Out:

Triggered when the mouse pointer leaves the element. It logs the event type (mouseout).

Mouse Over:

Triggered when the mouse pointer moves over the element. It logs the event type (mouseover).

Wheel:

Triggered when the user uses the mouse wheel over the element. It logs the event type (wheel).

Scroll:

Triggered when the page is scrolled. It logs the event type (scroll).

Touch Start:

Triggered when the user starts touching the element. It logs the event type (touchstart).

Touch End:

Triggered when the user ends touching the element. It logs the event type (touchend).

Touch Move:

Triggered when the user moves their finger over the element. It logs the event type (touchmove).

Drag:

Triggered when the user starts dragging the element. It logs the event type (drag).

Pointer Move:

Triggered when the pointer moves over the element. It logs the event type (pointermove).

Pointer Enter:

Triggered when the pointer enters the element. It logs the event type (pointerenter).

Pointer Leave:

Triggered when the pointer leaves the element. It logs the event type (pointerleave).
