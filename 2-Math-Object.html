What is Math in JavaScript?

The Math object in JavaScript provides built-in methods for performing various mathematical operations, from basic arithmetic to advanced calculations like trigonometry and logarithms.

Math Methods

  • Math.sin(x): Returns the sine of x (in radians).
  • Math.cos(x): Returns the cosine of x (in radians).
  • Math.tan(x): Returns the tangent of x (in radians).
  • Math.pow(x, y): Returns the value of x raised to the power of y.
  • Math.sqrt(x): Returns the square root of x.
  • Math.log(x): Returns the natural logarithm of x.
  • Math.exp(x): Returns e raised to the power of x.
  • Math.random(): Returns a pseudo-random number between 0 (inclusive) and 1 (exclusive).
