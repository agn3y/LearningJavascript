Ternary Operator

What is a Ternary Operator?

  • Ternary means three
  • Syntax: condition ? trueValue : falseValue


  • Example: 5 ? '' : 1
  • Explanation: In the example above, the condition is 5, which is a truthy value.
  • The ternary operator evaluates the condition. Since 5 is truthy, it moves to the trueValue, which is '' (empty string).
  • Despite '' being falsy, it is still the result of the ternary operator because it's the value for the true branch.
  • Thus, the result of 5 ? '' : 1 will be '' (empty string).

Another Example:

let result = (5 > 3) ? 'Greater' : 'Smaller';

This evaluates to 'Greater' because 5 > 3 is true.
