BOM in JavaScript

BOM stands for Browser Object Model. It refers to the objects provided by the browser to interact with the browser window and its components. Unlike the DOM, which deals with the structure and content of the webpage, the BOM allows developers to interact with the browser itself.

BOM - Browser Object Model

  • location
  • history
  • innerHeight
  • innerWidth
  • open()
  • close()
  • resizeBy()
  • resizeTo()
  • moveTo()
  • moveBy()
  • scrollBy()
  • scroll()
  • scrollTo()
  • print()
  • document

Key Features of BOM:

Window Object: The global object representing the browser window. It includes methods, properties, and events for controlling the browser window. 


// Example:
alert("This is part of the BOM!"); // Displays an alert box
console.log(window.innerWidth); // Logs the width of the browser window

Navigator Object: Provides information about the browser and device. 


// Example:
console.log(navigator.userAgent); // Outputs the browser's user agent string

Location Object: Represents the current URL and allows manipulation of it. 


// Example:
console.log(location.href); // Logs the current URL
location.reload(); // Reloads the page

History Object: Allows navigation through the browser's history. 


// Example:
history.back(); // Goes to the previous page in history
history.forward(); // Goes to the next page in history

Screen Object: Provides information about the user's screen. 


// Example:
console.log(screen.width); // Logs the width of the screen
console.log(screen.height); // Logs the height of the screen

Why is BOM Important?

The BOM gives developers tools to:

  • Interact with and control the browser environment.
  • Redirect or reload pages.
  • Access browser-specific information.
  • Perform actions like alerts, prompts, or console logging.

It helps in making web pages more interactive and adaptable to the user's environment.
