Functions

  1. Functions only execute when they are called
  2. Arguments are the values assigned to parameters
  3. With the return keyword in JavaScript, you can return almost anything—numbers, strings, objects, arrays, functions, booleans, and even null or undefined. Functions are very flexible in terms of what they can return, so you can use them to produce any kind of result you need
  4. However, you cannot directly return control structures like if or while loops themselves, but you can return values from them
  5. You can’t return multiple values directly (but you can return an array or an object containing multiple values)
  6. You can think of return as a mechanism that provides the result or output of a function

Syntax:

            
 function userNames(userNames, optional) { 
 console.log(`Hi, My name is ${userNames} and I am a Java-script Engineer.`);
 console.log(`I don't like Idli-Dhosa. Also, the ${optional} people.`);
 }
 userNames('agney','South-Indian');
 userNames();

Adding default parameters:

1st Way :

            
 function userNames(userNames = 'Unknown', optional = 'No specific') {
 console.log(`Hi, My name is ${userNames} and I am a JavaScript Engineer.`);
 console.log(`I don't like Idli-Dhosa. Also, the ${optional} people.`);
 }
 userNames('Agney', 'South-Indian');
 userNames();

2nd Way: Using (||) in the template string

            
                function userNames(userNames , optional ) {
                console.log(`Hi, My name is ${userNames || 'Unknown'} and I am a JavaScript Engineer.`);
                console.log(`I don't like Idli-Dhosa Also, the ${optional || 'No specific'} people.`);
                }
                userNames('Agney', 'South-Indian');
                userNames();

return

The return statement in a JavaScript function sends a value back to the caller and ends the function's execution. If omitted, the function returns undefined

Suppose we have a code like this:

 function addTwoNumbers(){
     console.log(5 + 6);  // Fixed syntax: changed curly braces to parentheses
 }
 addTwoNumbers();  // This will log 11 to the console

 // Code for returning two arguments
 function sumOfTwo(a, b){
     const x = 5;
     const y = 7;
     return x + y;  // This function will return the sum of x and y
 }

 const result1 = sumOfTwo(1, 22222);  // You can capture the return value here
 console.log(result1);  // Output: 12 (since 5 + 7 = 12)
 
 const result2 = sumOfTwo(12, 332);  
 console.log(result2);  // Output: 12 (same sum 5 + 7)

 const result3 = sumOfTwo(111, 2);
 console.log(result3);  // Output: 12 (same sum 5 + 7)

 const result4 = sumOfTwo(122, 34);
 console.log(result4);  // Output: 12 (same sum 5 + 7)

It will output 12, because the return value is always 5 + 7 (from the sumOfTwo function).

Return is also known as the result of a function, and in our case 12 is the result, not 11.

return keyword in a function essentially replaces the function call with its return value.
