Truthy and Falsy Values in JavaScript

In boolean contexts, the following values are considered falsy: 0, NaN, undefined, null, and empty strings (''). Any other value is considered truthy.

Note: Even a single space (' ') is truthy, unlike an empty string.

We can use the ternary operator to toggle values between true and false.
