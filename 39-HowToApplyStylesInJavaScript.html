Explanation of How to Apply Styles in JavaScript

1. Inline Styles: You can directly modify an element's style using the style property.
2. Using cssText: This lets you apply multiple styles at once using a string.
3. Adding/Removing Classes: You can add or remove predefined CSS classes using classList.
4. Using setAttribute: You can apply the style attribute directly to an element.

The styles will be applied here immediately.

        
            <script>
            function applyStyles() {
                const messageElement = document.getElementById('message');
                messageElement.style.color = 'blue';
                messageElement.style.fontSize = '24px';
                messageElement.style.backgroundColor = 'yellow';
                messageElement.style.padding = '10px';
            }
            window.onload = applyStyles;
            </script>
